Material

The range of materials that can be use in a 3D printer is amazing and continuing to be added to each day.

  • PLA-is the most common used in hobby grade printers and is made of corn starch.
  • ABS-is another common material used in hobby grade printers, but requires more heat
  • Composite-is a not so common but extremly strong and stiff material used to print objects that need to be strong
  • NinjaFlex-is not super popular but is a cool material that is flexible, some uses are phone cases and car tires
  • Woodfill, Bronzefill, CopperFill- are materials with Wood, bronze, or copper in them and are very cool to be used for decorative 3D prints

How it all Works

The Way 3d Printers work can seam very confusing while really it's quite simple. We have had some of the students understanding it by class 1.

  1. First you make the 3D design on TinkerCAD or any other CAD program of your choosing
  2. Then you import it into a software called a slicer which tells the printer how to print each layer of a 3D Print
  3. Next you move the sliced file to the 3D Printer where the printer prepare to print the file by making sure it is calibrated and heating up the extruder
  4. Finally the printer moves on the x and y axis to make each layer and the y to mave the extruder up and keep it at the next layer
  5. At last the print is finished and you can take it off the printer and use it!
